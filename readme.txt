Easy List/Bookmark Management for WordPress

== Description ==

Manage and create unlimited list and add items on list. Item can be any WordPress Post type, can be any Taxonomy even any other custom text with a title along with description.

[Try Demo](https://demo.pluginrox.com/wp-wishlist/)
`
    Username: wpwl
    Password: pluginrox
`

<strong>Plugin Features</strong>

*   Frontent list creation
*   Instantly create and add item to list
*   Support any post type and taxonomy
*   Can store any custom text with a description and use this as a to-do management
*   Developer friendly

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>WordPress Wishlist</strong>" activate it.<br />
